# Лабораторная работа 4

Структура проекта соответствует агентной реализации SIR-модели и дополнительным сценариям из лабораторной работы.

Основные сценарии:

- `scripts/sir_run_basic.jl`
- `scripts/sir_scan_beta.jl`
- `scripts/sir_migration_effect.jl`
- `scripts/sir_optimize_parameters.jl`
- `scripts/sir_visualize_dynamics.jl`
- `scripts/sir_heterogeneity_effect.jl`
- `scripts/sir_quarantine_scenario.jl`
- `scripts/sir_optimize_with_constraint.jl`
- `scripts/tangle.jl`

Артефакты:

- `plots/` — итоговые графики
- `data/` — CSV-файлы с результатами
- `scripts/<name>/` — clean `.jl`
- `notebooks/<name>/` — `.ipynb`
- `docs/<name>/` — `.qmd`
